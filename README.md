**Hangman Game**

A simple Hangman game built with React.The goal of the game is to guess the hidden word one letter at a time before your lives are over.

Do 7 letter words actually scare you? : https://hangman-gallows.vercel.app/

**Features:** 
Random word generation,
On-screen keyboard,
Correct and incorrect letter tracking,
Progressive hangman drawing,
Win and lose conditions
Responsive design,
Restart game functionality
